## 📖 Project Overview
This project performs **Exploratory Data Analysis (EDA)** on **BMW’s global automotive sales data** to uncover trends, patterns, and insights across regions and time periods.

The goal of this analysis is to support **data-driven decision-making** by understanding market performance, growth behavior, and sales dynamics.

---

## 🎯 Objectives
- Analyze global BMW sales trends over time  
- Compare regional and country-level performance  
- Identify top-performing markets  
- Detect seasonality and anomalies  
- Present insights using clear and effective visualizations  

---

## 📊 Dataset
- **Source:** https://www.kaggle.com/datasets/dmahajanbe23/bmw-global-automotive-sales 
- **Key Features:**
  - Revenue_EUR    
  - Model  
  - Units_Sold  
  - Avg_Price_EUR
  - Premium_Share
  - GDP_Growth
  - Fuel_Price_Index
